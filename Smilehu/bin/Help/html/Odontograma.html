Guía de Ficha Dental

Código

El código de la ficha dental es un identificador único que se asigna automáticamente al crear una nueva ficha dental. Este código se utiliza para realizar un seguimiento de la ficha dental en el sistema.

Nombre del Diente

En este campo, se debe ingresar el nombre del diente para el que se está creando la ficha dental. Este campo es obligatorio y solo se aceptan letras y espacios.

Prótesis

Este campo se utiliza para indicar si el diente tiene una prótesis o no. Se puede seleccionar una de las siguientes opciones: "No" o "Sí". Si se selecciona "Sí", se mostrará un campo adicional para ingresar el tipo de prótesis.

O si

Este campo se utiliza para ingresar información adicional sobre el diente. Por ejemplo, se puede utilizar para describir la condición o el tratamiento del diente.

Observaciones

Este campo se utiliza para ingresar observaciones adicionales sobre la ficha dental. Por ejemplo, se puede utilizar para describir la condición general del diente o para anotar detalles sobre el tratamiento planificado.

Estado

Este campo se utiliza para indicar el estado actual de la ficha dental. Se puede seleccionar una de las siguientes opciones: "Activo" o "Inactivo". Si se selecciona "Inactivo", la ficha dental ya no se mostrará en las búsquedas y los informes.

Guardar

Este botón se utiliza para guardar la ficha dental en el sistema. Después de hacer clic en el botón "Guardar", se creará una nueva ficha dental con la información proporcionada.
