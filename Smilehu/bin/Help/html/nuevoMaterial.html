Descripción de la Imagen

La imagen muestra una sección de una aplicación web para la gestión de materiales. La sección está dividida en dos partes: "Nombre Material" y "Nombre Proveedor". En la parte superior de la sección, hay dos campos de entrada de texto etiquetados como "Nombre Material" y "Nombre Proveedor", respectivamente, donde el usuario puede ingresar el nombre del material y el nombre del proveedor. Abajo de los campos de entrada de texto, hay un botón etiquetado como "Crear Material" que el usuario puede presionar para crear un nuevo material con el nombre y el proveedor ingresados. A la derecha de cada campo de entrada de texto, hay un ícono en forma de lápiz que indica que el usuario puede editar los campos.

Justo debajo de los campos de entrada de texto y el botón "Crear Material", hay una tabla que muestra el nombre del material, el nombre del proveedor y el precio del material. La tabla tiene una fila de encabezado y varias filas de datos. Cada fila de datos tiene tres columnas: una para el nombre del material, una para el nombre del proveedor y una para el precio. La tabla también tiene un botón etiquetado como "Crear Material" a la derecha de la fila de encabezado, que el usuario puede usar para crear un nuevo material y agregarlo a la tabla.
