Crear Paciente

Pasos para crear un nuevo paciente:

  1. Ingrese el nombre del paciente.
  2. Ingrese los apellidos del paciente.
  3. Ingrese la fecha de nacimiento del paciente.
  4. Ingrese el teléfono del paciente.
  5. Ingrese la dirección del paciente.
  6. Seleccione el género del paciente entre las opciones "Masculino" o "Femenino".
  7. Hag a clic en el botón Crear Paciente para guardar la información.
