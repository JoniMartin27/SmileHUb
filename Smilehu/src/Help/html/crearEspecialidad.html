Crear Especialidad

Para crear una nueva especialidad, siga los siguientes pasos:

  1. Ingrese el nombre de la especialidad en el campo de texto correspondiente.
  2. Seleccione el doctor correspondiente en el menú desplegable.
  3. Asegúrese de que el idioma seleccionado sea el correcto, ya que el sistema detectará automáticamente el idioma para la especialidad.
  4. Tenga en cuenta que la letra "ñ" se puede usar en el nombre de la especialidad.
  5. Si se produce un error, el sistema mostrará un mensaje con el código de error. Por ejemplo, el código de error "R-8" indica que el nombre de la especialidad ya existe en el sistema.
