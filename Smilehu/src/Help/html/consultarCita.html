Sección de Ayuda - Citas Médicas

Consultar Citas

  • Para consultar las citas, simplemente navegue a la sección "Consultar Citas".

  • Aquí podrá ver una lista de todas las citas registradas, junto con información relevante.

Crear Cita

  • Para crear una cita, vaya a la sección "Crear Cita".

  • Ingrese los detalles de la cita, como el ID de Cita, Nombre del Paciente, ID de Tratamiento, Observaciones, Fecha y Hora.

  • Haga clic en el botón "Crear" para guardar la cita.

  • La nueva cita se mostrará en la lista de citas.
