Ficha Crear Paciente

Pasos para crear un nuevo paciente:

  1. Ingrese el nombre del paciente.
  2. Ingrese la fecha de nacimiento del paciente.
  3. Ingrese el teléfono del paciente.
  4. Ingrese la dirección del paciente.
  5. Seleccione el género del paciente entre las opciones "Masculino" o "Femenino".
  6. Ingrese los apellidos del paciente.
  7. Haga clic en el botón Crear Paciente para guardar la información.
