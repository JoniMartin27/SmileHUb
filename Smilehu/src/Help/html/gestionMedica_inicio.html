Gestión Médica Inicio

Menú de Navegación

El menú de navegación permite al usuario acceder a diferentes módulos del sistema.

  • Especialidad: Este módulo permite gestionar las especialidades médicas del sistema. Se pueden crear nuevas especialidades, modificar las existentes y eliminar aquellas que ya no se necesiten.
  • Tratamiento: Este módulo permite gestionar los tratamientos médicos del sistema. Se pueden crear nuevos tratamientos, modificar los existentes y eliminar aquellos que ya no se necesiten. Además, el sistema permite buscar un tratamiento específico mediante un campo de búsqueda.
