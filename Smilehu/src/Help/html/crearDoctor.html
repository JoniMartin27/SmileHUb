Página de ayuda: Ficha Crear Doctor

Nombre

El nombre del doctor.

Apellidos

Los apellidos del doctor.

Especialidad

La especialidad médica del doctor.

Direccion

La dirección donde el doctor ejerce su práctica médica.

Genero

El género del doctor. Seleccionar "WALN" para "Woman", "Man", "Another gender", o "Not specified" (Otra genero o No especificado).

Crear Doctor

Un botón que se puede presionar para crear un nuevo perfil de doctor.
