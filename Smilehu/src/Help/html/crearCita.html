Página de ayuda: crear cita

Nombre Paciente

El nombre del paciente que recibirá el tratamiento.

Tratamiento

El tipo de tratamiento que recibirá el paciente.

Doctor

El nombre del doctor que realizará el tratamiento.

Material

Cualquier material o suministro que se necesite para el tratamiento. Esto podría incluir cosas como medicamentos, dispositivos médicos o otra equipos.

Observaciones

Cualquier nota o observación adicional sobre el tratamiento. Esto podría incluir cosas como la historia médica del paciente, cualquier instrucción especial para el doctor, o detalles importantes.

Buscar

Un campo de búsqueda donde puedes ingresar una palabra clave o frase para buscar una cita específica.

Crear Cita

Un botón que puedes presionar para crear una nueva cita.

Hora

Hora de la cita.

Fecha

Fecha de la cita.
